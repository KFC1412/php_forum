# EDUCN-Forum (php_forum_json_v0.2.0_t_260404)

## 简介

EDUCN论坛（EDUCN-Forum）是一个轻量级、高性能的 PHP 论坛程序，采用现代化架构设计，支持 MySQL 和 JSON 文件两种存储方式。v0.2.0_t_260404 版本在原有基础上进行了全面优化，新增完善的举报系统、申诉系统、邮件通知系统、系统日志管理、公告管理等实用功能，并大幅优化了 ICE 编辑器的用户体验。

本程序基于[LightForum](https://lightforum.org/)项目修改
LightForum GitHub 仓库：https://github.com/LightForum/LightForum

## 主要特性

### 核心功能
- **双存储模式**：支持 MySQL/MariaDB 数据库 和 JSON 文件存储
- **响应式设计**：完美适配桌面、平板和手机设备
- **伪静态支持**：SEO 友好的 URL 结构
- **富文本编辑器**：集成 iceEditor，支持图片、视频、代码等
- **缓存系统**：高效的缓存机制，提升系统性能
- **实时时间显示**：首页实时显示北京时间

### 用户系统
- **用户注册**：支持邮箱注册，邮箱验证功能
- **用户登录**：安全的登录机制，支持记住登录状态
- **找回密码**：通过邮箱重置密码
- **个人资料管理**：修改头像、昵称、邮箱、密码等
- **头像系统**：支持本地上传和 URL 链接设置头像
- **用户等级**：基于发帖数的用户等级系统
- **用户状态**：正常、受限、封禁等状态管理
- **积分系统**：用户行为获得积分，支持积分规则配置
- **经验值系统**：基于积分计算经验值，决定用户等级
- **等级徽章**：根据等级授予不同徽章，展示用户成就
- **积分记录**：详细记录用户积分变动历史
- **实名认证**：支持用户提交实名认证，管理员审核
- **专业认证**：支持专业领域认证，提升用户可信度
- **认证管理**：后台查看和管理认证申请

### 内容管理
- **分类管理**：多级分类结构，支持置顶分类
- **主题发布**：富文本编辑，支持图片、视频、代码等
- **主题编辑**：编辑已发布的主题
- **主题删除**：删除主题及其所有回复
- **回复功能**：支持引用回复、回复通知
- **主题锁定**：管理员可锁定主题，禁止回复
- **主题置顶**：管理员可置顶重要主题
- **主题推荐**：管理员可推荐优质主题
- **主题隐藏**：自动或手动隐藏违规主题
- **搜索功能**：全文搜索主题和回复
- **浏览统计**：记录主题浏览次数
- **内容互动**：支持点赞、收藏主题和回复
- **标签系统**：为主题添加标签，方便分类和搜索
- **热门标签**：展示热门标签云
- **标签搜索**：通过标签查找相关内容
- **内容审核**：自动审核和人工审核相结合
- **敏感词过滤**：自动检测和过滤敏感内容
- **垃圾内容检测**：识别和拦截垃圾信息
- **违规记录**：记录用户违规行为和处理结果
- **专题管理**：创建和管理内容专题
- **精选内容**：管理员精选优质内容

### 举报系统
- **多类型举报**：支持举报主题、回复、用户
- **举报理由**：用户可填写详细举报理由
- **举报管理**：后台查看所有举报记录

- **举报阈值设置**：

  - 主题达到封禁阈值：自动隐藏主题

- **自动恢复**：举报和申诉都删除后自动恢复正常状态

## 安装说明

### 环境要求
- PHP 7.4 或更高版本
- MySQL 5.6+ 或 MariaDB 10.2+（如使用数据库存储）
- Apache 或 Nginx 服务器
- 支持 PHP 的邮件发送功能（SMTP）

### 安装步骤
1. 上传所有文件到服务器
2. 访问 `http://你的域名/install/` 进入安装向导
3. 按照提示完成安装
4. 安装完成后删除 `install` 目录
5. 登录后台进行初始配置

### 目录结构
```
x:\www\
├── admin\              # 后台管理
├── includes\           # 核心文件
├── install\            # 安装程序
├── storage\            # 数据存储
│   ├── cache\         # 缓存文件
│   ├── json\          # JSON 数据文件
│   ├── mail_queue\    # 邮件队列
│   └── uploads\       # 上传文件
├── templates\          # 模板文件
├── 60s\              # 每日60秒热点
├── assets\            # 静态资源
└── *.php              # 主要页面文件
```

## 使用说明

### 用户操作
- **注册**：点击注册按钮，填写用户名、邮箱、密码
- **登录**：使用用户名或邮箱登录
- **发布主题**：登录后点击"发布新主题"
- **回复**：在主题页面底部回复框中输入内容
- **举报**：点击主题或回复旁的举报按钮
- **申诉**：被封禁用户或被隐藏主题作者可提交申诉

### 管理员操作
- **登录后台**：访问 `admin.php`，使用管理员账号登录
- **用户管理**：查看、编辑、删除用户，管理用户状态
- **内容管理**：管理分类、主题、回复
- **举报处理**：查看举报记录，进行处理
- **申诉审核**：查看申诉记录，进行审核
- **系统设置**：配置站点、邮件、举报等设置
